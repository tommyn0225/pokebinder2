title: Quickstart
description: Call every Binder API endpoint end to end in one sitting — public binder, its history, a trade list, catalog search, a card, and card history.
---

This walks through every endpoint in order. Each step is a single `curl` against
`https://v0-pokebinder.vercel.app` — no keys, no headers, no request body, no
setup. Run them top to bottom.

## 1. Confirm the API is live

The API is public and hosted, so there's nothing to install. Confirm it responds:

```bash
curl https://v0-pokebinder.vercel.app/api/v1/openapi.json
```

You should get the OpenAPI document back as JSON.

<Note>
  Prefer to run Binder yourself? Start it with `npm run dev` and use
  `http://localhost:3000` as the base URL instead — every command below works the
  same way.
</Note>

## 2. Make a binder public

The binder endpoints only return **public** binders. In the app, open a binder
and choose **Make public**, then copy its share link — it looks like
`/b/<binder-id>`. The `<binder-id>` is a UUID; that's what you pass below.

Trade lists need no such step: they are **always public**. Open **View trades**
and copy the share link `/t/<token>`; it works as soon as you have it.

Throughout this guide the placeholder binder id is
`3c90c3cc-0d44-4b50-8888-8dd25736052a` and the placeholder trade token is
`a1b2c3d4-5e6f-4a7b-8c9d-0e1f2a3b4c5d`. Substitute your own.

## 3. Get the public binder

```bash
curl https://v0-pokebinder.vercel.app/api/v1/binders/3c90c3cc-0d44-4b50-8888-8dd25736052a
```

```json

  "binder": { "id": "3c90c3cc-0d44-4b50-8888-8dd25736052a", "name": "Modern staples", "game": "mtg" },
  "card_count": 2,
  "total_usd": 5.0,
  "holdings": [
    {
      "card_id": "e3285e6b-3e79-4d7c-bf96-d920f973b122",
      "name": "Lightning Bolt",
      "game": "mtg",
      "set_name": "Magic 2010",
      "set_code": "m10",
      "collector_number": "146",
      "rarity": "common",
      "image_url": "https://cards.scryfall.io/normal/front/e/3/e3285e6b-3e79-4d7c-bf96-d920f973b122.jpg",
      "quantity": 2,
      "finish": "nonfoil",
      "for_trade": true,
```

A missing or private binder returns `404` with the standard
[error envelope](/reference/conventions#errors):

```json
{ "error": { "code": "not_found", "message": "Binder not found or not public" } }
```

## 4. Get the binder's value history

Binder's own daily snapshots — not upstream history. One point per day the
binder has been tracked.

```bash
curl https://v0-pokebinder.vercel.app/api/v1/binders/3c90c3cc-0d44-4b50-8888-8dd25736052a/history
```

```json
{
  "binder_id": "3c90c3cc-0d44-4b50-8888-8dd25736052a",
  "history": [
    { "day": "2026-07-16T00:00:00+00:00", "total_usd": 4.98 },
    { "day": "2026-07-17T00:00:00+00:00", "total_usd": 5.0 }
  ],
  "attribution": "…"
}
```

## 5. Get a public trade list

```bash
curl https://v0-pokebinder.vercel.app/api/v1/trades/a1b2c3d4-5e6f-4a7b-8c9d-0e1f2a3b4c5d
```

```json
{
  "total_cards": 1,
  "cards": [
    {
      "card_id": "base1-4",
      "name": "Charizard",
      "game": "pokemon",
      "set_name": "Base Set",
      "set_code": "base1",
      "collector_number": "4",
      "rarity": "Rare Holo",
      "image_url": "https://v0-pokebinder.vercel.app/api/cards/image?id=base1-4",
      "quantity": 1,
      "finish": "nonfoil",
      "price": { "usd": 300.0, "usd_foil": null, "eur": null }
    }
  ],
  "attribution": "…"
}
```

Every card in a trade list is for trade, so these cards omit the `for_trade`
flag that binder holdings carry.

## 6. Search the card catalog

The catalog needs no ids — it's every card tracked in Binder, in one normalized
model across all three games:

```bash
curl "https://v0-pokebinder.vercel.app/api/v1/cards?q=charizard&game=pokemon"
```

```json
{
  "count": 1,
  "has_more": false,
  "next_cursor": null,
  "cards": [
    {
      "card_id": "base1-4",
      "name": "Charizard",
      "game": "pokemon",
      "set_name": "Base Set",
      "set_code": "base1",
      "collector_number": "4",
      "rarity": "Rare Holo",
      "image_url": "https://v0-pokebinder.vercel.app/api/cards/image?id=base1-4",
      "price": { "usd": 300.0, "usd_foil": null, "eur": null }
    }
  ],
  "attribution": "…"
}
```

`game` and `q` are optional; `limit` sets the page size (default 50, max 100).
`count` is the number of cards in **this page**, not the catalog total. When
`has_more` is true, pass `next_cursor` back as `?cursor=…`. The full loop is in
[Search and paginate the catalog](/guides/search-and-paginate).

## 7. Get one card with its price history

Use a `card_id` from the search above. The history is Binder's own daily
snapshots, one point per day, all-time:

```bash
curl https://v0-pokebinder.vercel.app/api/v1/cards/pokemon/base1-4
```

```json
{
  "card": { "card_id": "base1-4", "name": "Charizard", "game": "pokemon", "set_name": "Base Set", "set_code": "base1", "collector_number": "4", "rarity": "Rare Holo", "image_url": "https://v0-pokebinder.vercel.app/api/cards/image?id=base1-4", "price": { "usd": 300.0, "usd_foil": null, "eur": null } },
  "history": [
    { "day": "2026-07-16", "price_usd": 295.0 },
    { "day": "2026-07-17", "price_usd": 300.0 }
  ],
  "attribution": "…"
}
```

A card no Binder user tracks returns `404`:

```json
{ "error": { "code": "not_found", "message": "Card not found in catalog" } }
```

## 8. Get a card's price history over a window

Same history, bounded by `days` (default 30, max 365):

```bash
curl "https://v0-pokebinder.vercel.app/api/v1/cards/pokemon/base1-4/history?days=7"
```

```json
{
  "game": "pokemon",
  "card_id": "base1-4",
  "days": 7,
  "history": [
    { "day": "2026-07-16", "price_usd": 295.0 },
    { "day": "2026-07-17", "price_usd": 300.0 }
  ],
  "attribution": "…"
}
```

## 9. Try it with zero setup

Don't want to run the app? The docs **API reference** tab has an interactive
playground, and the `/api/demo/cards` endpoint pulls real catalog cards without
any setup — capped at 3 pulls per IP per day:

```bash
curl "https://v0-pokebinder.vercel.app/api/demo/cards?q=charizard&game=pokemon"
```

```json
{
  "count": 1,
  "demo": { "limit": 3, "remaining": 2 },
  "cards": [ { "card_id": "base1-4", "name": "Charizard", "game": "pokemon", "…": "…" } ],
  "attribution": "…"
}
```
